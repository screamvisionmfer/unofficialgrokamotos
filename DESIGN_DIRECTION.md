# Unofficial Grokamotos Pip-Boy Redesign

## Project

Unofficial Grokamotos Pip-Boy redesign.

## Visual Target

- Fallout 3 / New Vegas Pip-Boy interface.
- Black background with phosphor green CRT UI.
- Vault-Tec terminal startup feeling.
- Green scanlines, glow, slight blur, screen noise.
- Chunky retro terminal typography.
- Thin outlined UI panels.
- Tab navigation like Pip-Boy: MAIN / GALLERY / DRB WALLET / CONTACTS / GAME.
- Meme character presence, but not too much text.
- Balanced tone: 50% serious Fallout homage, 50% Grokamotos meme energy.

## Must Preserve

- Existing boot menu idea.
- Rotating/changing logos/images in boot menu.
- Current sounds.
- Current cursor design.
- Current wallet API logic.
- Current gallery logic.
- Current achievements/localStorage keys.

## Future Features

- Vault-Boy style Grokamotos character.
- VATS-like hover zones on character:
  - Head = hat traits count.
  - Eyes = eye traits count.
  - Mouth = mouth traits count.
  - Body = body traits count.
  - Back = back accessories count.
- Color scheme switcher:
  - Green.
  - Amber.
  - Red emergency.
  - Blue CRT.
- Future hacking mini-game inspired by Fallout terminal password hacking.

## Avoid

- Modern SaaS UI.
- Colorful cyberpunk.
- Glassmorphism.
- Too much dense text.
- Changing content/logic before layout is stable.
- Copyrighted Fallout text/logos copied directly.

## Redesign Plan

1. Global Pip-Boy skin through CSS only.
2. Boot menu visual reskin.
3. Main layout into Pip-Boy tabs.
4. Hero as STATUS screen.
5. Gallery as DATA/archive screen.
6. Wallet as DRB WALLET meter screen.
7. Contacts screen.
8. Future GAME screen.
